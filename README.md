# Backbone.DatepickerView

Bootstrap style datepicker backbone view

## Usage

```js
var view = new Backbone.DatepickerView({
  el: $('.input-date'),
  placeholder: 'Datepicker'
});
```

## Options

```coffee
defaults:
  dateFormat: 'yyyy-MM-dd'
  name: 'date'
  span: 'span2'
  placeholder: 'Date'
  defaultValue: null
  settings:
    pickSeconds: false
```
