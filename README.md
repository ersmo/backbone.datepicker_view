# Backbone.DatepickerView

Bootstrap style datepicker backbone view

## Usage

var view = new Backbone.DatepickerView({
  el: $('.input-date'),
  placeholder: 'Date Picker'
});

## Options

```coffee
defaults:
  dateFormat: 'yyyy-MM-dd'
  name: 'date'
  span: 'span2'
  placeholder: 'Date pick'
  defaultValue: null
  settings:
    pickSeconds: false
```
